# 🛍️ Customer Behavior Analysis Project

## 📌 Overview

This project focuses on analyzing customer shopping behavior to uncover actionable insights that can help businesses improve sales performance, customer engagement, and decision-making.

The analysis covers customer demographics, purchasing patterns, product categories, and subscription behavior.

---

## 🎯 Business Problem

A retail organization wants to understand:

* How customer behavior varies across demographics and categories
* What factors influence purchasing decisions
* How to improve customer engagement and loyalty
* How to optimize marketing and product strategies

---

## 🔄 Project Workflow

1. **Data Collection & Import**

   * Imported raw dataset into Python

2. **Data Cleaning & Preparation**

   * Handled missing values

3. **Exploratory Data Analysis (EDA)**

   * Identified trends, patterns, and anomalies

4. **Data Modeling (SQL)**

   * Loaded data into SQL database
   * Performed queries for business insights

5. **Dashboard Creation**

   * Built interactive dashboard in Power BI
   * Visualized KPIs and trends

6. **Insights & Reporting**

   * Derived insights and recommendations
   * Created presentation for stakeholders

---

## 📊 Key Insights

* ~92% customers are **non-subscribers** → Opportunity to increase subscriptions
* **Clothing category** generates highest revenue and sales
* **Adults & Young Adults** are the most valuable customer segments
* Average purchase value: **$59.76**
* Average customer rating: **3.75**
* Categories like **Outerwear** underperform and need attention
